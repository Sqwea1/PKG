# Image Information Reader

## Описание

Image Information Reader - это приложение для анализа графических файлов, которое извлекает основную информацию об изображениях. Программа поддерживает все основные форматы растровых изображений и предоставляет подробную информацию о каждом файле.

## Возможности

- Поддержка форматов: JPG, JPEG, GIF, TIF, TIFF, BMP, PNG, PCX
- Автоматическое сканирование выбранной папки
- Отображение информации об изображениях:
  - Имя файла
  - Размер в пикселях
  - Разрешение (DPI)
  - Глубина цвета
  - Метод сжатия
- Предпросмотр изображений
- Открытие файлов в стандартном просмотрщике двойным кликом
- Многопоточная обработка больших объемов файлов

## Требования

- Windows 7/8/10/11
- Для исходного кода: Python 3.6+

## Установка и запуск

### Способ 1: Запуск EXE файла (рекомендуется)

1. Скачайте файл `ImageInfoReader.exe`
2. Запустите исполняемый файл
3. Программа готова к работе

### Способ 2: Запуск из исходного кода

1. Установите Python 3.6 или выше
2. Установите необходимые библиотеки:
   ```bash
   pip install Pillow
   ```
3. Скачайте файл `image_analyzer.py`
4. Запустите программу:
   ```bash
   python image_analyzer.py
   ```

## Использование

1. Нажмите кнопку "Browse" для выбора папки с изображениями
2. Программа автоматически начнет сканирование выбранной папки
3. Информация об изображениях отобразится в таблице
4. Для просмотра изображения выберите файл в таблице - предпросмотр появится справа
5. Для открытия файла в стандартном просмотрщике сделайте двойной клик по строке в таблице

## Техническая информация

### Используемые библиотеки
- **Pillow** - работа с изображениями и метаданными
- **Tkinter** - графический интерфейс
- **Threading** - многопоточная обработка

### Поддерживаемые форматы
- JPEG/JPG (сжатие JPEG)
- GIF (сжатие LZW)
- TIFF (различные методы сжатия)
- BMP (обычно без сжатия)
- PNG (сжатие Deflate)
- PCX (сжатие RLE)

### Извлекаемая информация
- Размер изображения в пикселях
- Разрешение (DPI)
- Глубина цвета (биты на пиксель)
- Метод сжатия
- Информация о цветовом пространстве

## Производительность

- Оптимизированная обработка больших объемов данных (до 100,000 файлов)
- Многопоточная архитектура для избежания блокировки интерфейса
- Тестировано на папках с 600+ JPEG файлами общим объемом 2ГБ
