# Color Converter

## Описание проекта

Color Converter — интерактивное приложение для преобразования цветов между различными цветовыми моделями: **CMYK, LAB и RGB**. Позволяет выбирать цвет и интерактивно изменять его, отображая составляющие во всех трех моделях одновременно.

## Основной функционал

- **Конвертация между цветовыми моделями**: CMYK ↔ LAB ↔ RGB
- **Три способа выбора цвета**:
  - Диалоговое окно выбора цвета
  - Ползунки для каждой компоненты
  - Ручной ввод значений
- **Автоматический пересчет**: при изменении любой компоненты все остальные модели обновляются в реальном времени
- **Визуальное отображение** текущего цвета для каждой модели

## Технические требования

- **Операционная система**: Windows 7 и выше

## Установка и запуск

###Исполняемый файл
1. Скачайте файл `pkg_1.exe`
2. Запустите двойным щелчком мыши

## Руководство пользователя

### Интерфейс приложения

- **Кнопка "Выбрать цвет из палитры"** - открывает стандартный диалог выбора цвета
- **Цветовые поля** - отображают текущий цвет для каждой модели
- **Текстовые поля** - для ручного ввода значений компонентов
- **Ползунки** - для плавного изменения значений

### Работа с приложением

**Выбор цвета через диалог:**
1. Нажмите "Выбрать цвет из палитры"
2. Выберите цвет в диалоговом окне
3. Нажмите "OK"

**Изменение с помощью ползунков:**
- Перемещайте ползунки для изменения компонентов
- Диапазоны значений:
  - CMYK: 0-100%
  - LAB: L(0-100), A(-128...+127), B(-128...+127)
  - RGB: 0-255

**Ручной ввод значений:**
- Введите значения в текстовые поля через запятую
- Пример: 255, 128, 0 для RGB
- Нажмите Enter для применения

## Цветовые модели

### CMYK (Cyan, Magenta, Yellow, Key)
- Субтрактивная модель для печати
- Компоненты: C, M, Y, K (0-100%)

### LAB
- Модель, основанная на восприятии цвета человеком
- Компоненты:
  - L (Lightness) - яркость (0-100)
  - A - от зеленого к красному (-128...+127)
  - B - от синего к желтому (-128...+127)

### RGB (Red, Green, Blue)
- Аддитивная модель для экранов
- Компоненты: R, G, B (0-255)
